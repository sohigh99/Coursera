Purpose

This document describes how to turn off workflow messages.

Questions and Answers

Question 1:  With workflow in-activated (not activated), why does the message "your changes have been submitted for approval and are pending review" show when an attempt is made to terminate an employee in the employee information application (P0801)?

Answer 1:  Access the Object Management Workbench (OMW in the fastpath).  Search on Workflow/Process = TERM*.  Note that there are a number of termination workflows.

If workflow for employee termination is not desired, disable all TERM workflows:

  • Go into the design form for each TERM workflow to select Disable workflow. Do this also for Workflow/Process= EEMAST1 if changes to the employee master also do not need approval.

  • When complete, access the Process Task Monitor P98860. Search on any active workflows. Highlight the grid row of the pending workflow for an employee and take the Row Exit to Terminate. This is terminating the workflow(s) previously submitted.

Question 2:  How to turn off workflow?

Answer 2:

  1. Go to OMW
  2. Click on the Search Tab
  3. Category = Workflow
  4.     Search Type defaults
  5.     Search = workflow process (ex: TERM*)
  6. Click Find
  7. All of the workflow processes on the right hand side need to be set to Inactive
  8. Click on the 1st option
  9. Click on the “Add Object or User to Project” Button dragging to the left hand side
  10. Highlight the object and click on the Design button in the middle
  11. Click on the Design Tools Tab
  12. Click on the Change Workflow Status Button
  13. Hit OK
  14. Once they are all changed to Inactive, go to P98860
  15. Process = workflow process (ex: TERM*)
  16. Click Find
  17. If any come up as active, take the row exit to Terminate
  18. Run Purge Completed Tasks (R01131P)
  19. Run Purge Completed Processes (R98860P)

Question 3:   Does System allow to Automatically Pay the Unpaid leave balance during the final payroll of the Employee when all the Termination Workflow are Enabled?

Answer 3: System will not calculate unpaid time automatically for Terminations. Unpaid Leave Balance should be calculated manually.
