Field Available In F4111

Purpose of Document

Is there any available column to use from F4111 for recording a certain process?

I have checked exiting tables and mapping, which appears,
- There are two columns which are not mapped are,
LPNU License Plate Number
IPCD As Of Posted Code : but this column is used for As-Of processing and reconciliation so do not use this column
- There are column which are dependent on Agreement management
DMCS Agreement Supplement
- There are specific usage (like Warehouse and production number) these are depending on your implementation
