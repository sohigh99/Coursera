Q1> What is UNSPSC?

A1> The United Nations Standard Products and Services Code® (UNSPSC®), managed by GS1 US™ for the UN Development Programme (UNDP), is an open, global, multi-sector standard for efficient, accurate classification of products and services. Refer to https://www.unspsc.org/ for detail.

The UNSPSC is an 8-digit code; comprised of four parts of (each part has two codes). The codes are hierarchical, similar to an outline. As you get deeper in the outline, there is more detail. The most detail is at the Commodity level (include both goods and services). The four parts are:
1 .Segments (general)
2. Family
3. Class
4. Commodity (detail)

Example:

Q3> How to import to the EnterpriseOne?

A3> As the implementation guide reads, P40241 (Work With UNSPSC Commodity Code) enables a user to upload excel. Note that the format given by UNSPSC may vary version by version. To access it,

  1. Go to unspsc.org
  2. (Tab) CODESET-DOWNLOADS
  3. Choose the version your business needs

 

Q4> How E1 handles UNSPSC for the service (Non-stock item)?

A4> As the implementation guides read, UNSPSC can be associated only through commodity code for non-stock item even when you do not have the item master (F4101) for the service you want to purchase.

Q6> Known Issue?

A6> Length limitation of F40241.UNSPSCDS (UNSPSCDescription) is defined with 40 digits including blanks. Refer to Enh 27891744 - DESCRIPTION SIZE TOO SMALL FOR UNSPSC COMMODITY CODES ( ALIAS UNSPSCDS )

 

Q7> Why the code given by UNSPSC is different from the format of F40241 which has only two columns (UNSPSC code and Description)?

A7> Group UNSPSC by Segment, Family, Class, and Commodity. As the question 1 indicates Segment, Family, and Class repeats for multiple Commodity. So the E1 item number and UNSPSC can have many to 1 relationship through P4101.

 

Q8> Where to build the relationship between UNSPSC code and E1 Item Number?

A8> Go to P4101 (Item Master), (tab) Additional Information. Specify the commodity code and UNSPSC. Note that currently E1 reports does not print UNSPSC.
