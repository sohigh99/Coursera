R43500 and language override - Case Study

Purpose of Document

This document discusses how languages can be handled in print R43500 (Print Purchase Order). It also provides information on tables that contains language settings.

  • Table of Contents:
    • Business Scenario
    • Layout of R43500

Business scenario:
There is a business requirement to print Purchase Order in Simplified Chinese. How can this be fullfilled?

  • Title, column heading has to be in Simplified Chinese
  • Supplier's address has to be written in Chinese
  • Item Description has to be written in Chinese

First of all, verify basic setup

  • - (Package) Have they installed language pack for Chinese Simplified and do they have package for language 'CS - Chinese Simplified'
  • - (Fonts) Have they installed valid fonts and is it defined correctly in P98980 (Language Fonts). For this example, STSong-Light_Acro
  • - Have they set valid language for the supplier in Address Book Master (F0101.LNGP)
  • - Have they populated UDC codes for language 'CS' (in F0005D) and Item Description in Chinese (in F4101D)


Note address lines will be printed as it is stored in F0116 (Address By Date) except Country Description (Alternatively, you may populate F4006 (Order Address Information) order by order from Purchase Order Header). For further detail, refer:-

Note:
  • Business view for R43500 is V43500JA which is joined business view between F4301 (Purchase Order Header) and F4311 (Purchase Order Detail)
  • There is NOT repository to store address lines information (F0116) per language. Hence, either you change Address Lines description as your business requirement or override it again through P4006 (Order Address Information)
  • Ref #1. ProcOptGlobalPrintMessage3 is determined by BSFN N4002340 - CheckIfNotesPrintOnReport (F4017 Check If Notes Print On Report). Table F4016 can store data for multiple languages
  • Supplier's Preferred Language is F0101.LNGP where AN8 = SHAN
  • In case you need to set a certain constant different from translation, you may override is through Language Override for a specific version
