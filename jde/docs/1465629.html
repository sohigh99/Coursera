Layout of R3483 (Master Planning Schedule - Multiple Plant):

  • Report Level
    • Initialize Report: Report Level Initialization including parallel processing
  • Section Level
    • Read Item Branch Records (Read Item Branch Records and Build UI)
      • Initialize Section: Make data selection against F4102 (Item Branch File)
      • Do Section: Edit, Validate, Create Cache for message
      • End Section: Write messages based on cache created during Do Section
    • (Conditional) Write Time Series for CoBy Products
      • Do Section
  • Report Level
    • End Report: Terminate cache, close tables and ends parallel processing
Note:
  • Do Section events read data from target table row by row whereas all other events are getting called only once in a routine
  • Conditional section [Write Time Series for CoBy Products] can be called when item handled in [Read Item Branch Record] is not a process item (then check if the item is a Co-Product). If it is co-by product and it is consolidation mode
  • Report level event can be accessed via Report Design Aid > File > Event Rule Operations > Edit Report Level Event Rules

1.    Open cursor, Reset cursor.
