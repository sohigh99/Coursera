Goal

Provide examples for Leave Balance Rollover DBA's (Deduction, Benefit, Accruals). The DBA codes are defined using PDBA Setup (P059116) on the Pay/Deductions/Benefits Setup menu (G05BD4). Calculation Tables (P059021) and Basis of Calculation (P059118) can be accessed from the menu or from row exits within PDBA Setup. Advanced DBA Information (P059117) and Rollover Setup (P07RSW) can be accessed from row exits within PDBA Setup.

Solution

The following tables contain examples of Leave Balance Rollover DBAs (Deduction, Benefit, Accruals) and pay type setup. The DBA codes are defined using PDBA Setup (P059116) on the Pay/Deductions/Benefits Setup menu (G05BD4). Calculation Tables (P059021) and Basis of Calculation (P059118) can be accessed from the menu or from row exits within PDBA Setup. Advanced DBA Information (P059117) and Rollover Setup (P07RSW) can be accessed from row exits within PDBA Setup.

Rollover Setup (P07RSW)

FieldValue
Benefit/Accrual Type V V (Vacation Accruals) or S (Sick Accruals)
Rollover Table 00002
Leave Balance Rollover Date Code Defaults from Advanced DBA Information (P059117)
Related PDBA* 811 (Vacation)
Related PDBA 8015 (Vacation Accrual)

* Do not associate a previously used pay type or accrual to more than one accrual; doing so yields unreliable available amounts.
Note: There is no ITD Limit in a two-DBA setup.

Leave Balance Lost DBA Setup

The Fiscal and Anniversary Year History (F06147) table records the "lost" amount. This DBA is not required, but tracks all amounts lost by employees that do not use their accrued time.

FieldValue
DBA Code 9001-9999 recommended
DBA Type A (Accrual)
Paystub Tex Bal Lost
Print Method N (Do not print on pay stub)
Source of Calculation G (Current Period)
Method of Calculation $ (Flat Amount)
Amount Rate 1 Blank
Pay Period of the Month N (No) for all periods
