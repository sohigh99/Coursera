Purpose
Scope

Purpose

This document provides a list of standard EnterpriseOne Financials UBE reports that must be run through a single-threaded job queue. Some exceptions may be noted. A single threaded UBE means the UBE cannot have more than one instance running at the same time. If a UBE is not listed in this document, it most likely runs in a multi-threaded queue.

NOTE: For UBE reports not specifically listed in this document, when deciding to run concurrent versions and/or queues, processing options and data selection should be used to prevent selection of the same records. It is crucial that the data selections of the individual versions not overlap to avoid data corruption or data inconsistency.

Scope

This document is intended for the system administrator who will be defining the job queues within the EnterpriseOne environment and maintaining and troubleshooting the execution of the reports within the environment.

Details

This is a list of standard Financial UBE reports that must run in a single-threaded job queue(s) defined in EnterpriseOne.

When two distinct submissions of this UBE are waiting for one another to complete its processing of a table, a deadlock may occur. It is recommended that these two processes are not run concurrently.

10

General Accounting

R10861

Delete Prior Consolidation program

Clears the account balances from the F0901 and F0902 tables so that when you run subsequent consolidations, the balances do not include amounts from previous consolidations. This program also deletes prior consolidation records from the F0911 table if journal entries were created to consolidated accounts.

10

General Accounting

R10862

Refresh Consolidation program

This program creates additional records in the F0902 table for the fictitious business units, accounts, and account balances within the fictitious company that you specify in the processing options. But, if you do not delete the information in the consolidation database before you run a new consolidation, the system displays an error message that balances exist, and you must run the R10861.
