Purpose of Document

This note is to explain how EnterpriseOne control process a certain process within Distribution and Manufacturing. For the first part this document is to explain Vertical Feature

This document is to explain Vertical Features in Distribution Module when you need to activate it or debug possible issue caused by setup by indicating related objects and applications. Currently, UDC 40/VF - 'Vertical Features' reads as below,

Note:
  • Since it affect across system globally we call it as vertical
  • Codes written above reflects enhancement in EnterpriseOne
  • Some code may repeate in F99410 (OneWorld System Control File)
  • You need to reset Database Cache because Table F99410 (OneWorld System Control File) and F0004/F0005 (UDC) are a member of F98613 (Business View Environmental Server) which can be viewed through P98613 (Work With Database Caching)
  • CPG stands for Consumer Packaged Goods
  • Description 01 does not affect functionality
  • DO NOT add if your business does not require it as this may cause performance issue

Business Function: B4002670 (VerticalFeature - Vertical Features)

  • Function: This business function is to determine whether a certain vertical feature is written/enabled by checking UDC 40/VF and/or F99410
  • Data Structure: D4002670 - Vertical Feature

Structure Member Name Data Item Required I/O DD Alias
szVerticalEnhancement CacheActionCode N (I) CACTN Input value
01, 02, 03 and 42 are always 1
cEnhancementExistance ModuleExistance N (O) MEOW Return
0 - No Enhancement
1 - Enhancement Exist

  • Detail routine:
    • If input parameter szVerticalEnhancement is IN ("01", "02", "03", "42"), it sets cEnhancementExistance = 1 without checking F99410
    • If the input value is not IN ("01", "02", "03", "42") then it looks for data from F99410 based on DD Alias name defined
    • Since input parameter is 01, 02, 03 or 42 it never goes to routine to check F99410 as it does not have routine for that
    • So it is important to know that Log (voucher) is not handled by B4002670

Note: CPG Enhancement is available from EnterpriseOne release 8.9 and above. To have better understanding background of these enhancements refer value positioning documents or net change documentation
