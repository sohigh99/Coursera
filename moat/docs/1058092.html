How the R07500 Update Tax Area Table is setup and run. What does it do?


Solution

Vertex, Inc. periodically provides updates to ensure the accuracy of your organization's payroll calculations. You can automatically update the tax area information that is stored in the F069016 table with the tax information that is included in the updates that you receive from Vertex, Inc.

Access HRM Common Display Settings (P05004)

  1. Search for Data Item HRGEO
  2. If set to No, set to Yes
  3. If HRGEO is not listed, add it


Access Tax Area Table Setup (P07500) on the Taxes and Insurance Setup menu (G07BPTI4)

  1. Select the Show All radio button
  2. Select the states and/or CN for Canada to process by clicking in the attachments column to place a green check mark
  3. Click OK

This acts as data selection for the R07500
Federal tax area will always be updated. There is no selection for this tax area.

Select Update Tax Area Table (R07500) on the Taxes and Insurance Setup menu (G07BPTI4)

  1. Review the processing option and set as desired. The F069016 referenced on the processing option is the Payroll Tax Area Profile a.k.a. Tax Area Information
  2. Submit the R07500 - A pdf file will be generated listing the state, the geocode and the number of records updated for that state along with a total record count. This record count will not match the number of state records in the Tax Area Information table as the record count includes, but does not display separately, the Federal records.


This report updates the Tax Area Information application/Payroll Tax Area Profile table (P069012/F069016) and the Compensation Overrides application/table (P0701731/F0701731).

The R07500 will only create/update tax areas/tax types that generate a tax.
This job can take a very long time to process depending on the number and/or size of the states selected.
If the geocoder functionality is not being used or the HRGEO was set to no initially, return to the HRM Common Display Settings (P05004) and set the data item back to "No".

NOTE: The Geocode Databases must be updated on the Vertex Install in order for the R07500 to run correctly.
